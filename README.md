### BACKEND development

1.create a git repo
    git init
2. Add ,gitignore file

3. create .env file for environment variabes & read data from .env with .env module
    npm i dotenv

4. Generate package.json
    npm init -y
5. create express application
    npm i express

6. connect to database
    install mongoose npm i mongoose

7. Add middlewares( bady parser,err handling middlewares) 

8. Design Schema and create models

9. Design APIs for the resources
